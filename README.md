# catchat

`catchat` — веб-приложение соцсети для людей 15–30 лет, созданное для поиска друзей по интересам из разных стран.

Пользователи могут общаться, находить сообщества, публиковать контент и знакомиться в интерактивных форматах. Фото в публикациях проходят модерацию перед появлением в приложении.

## Технологический стек (frontend)

- Next
- Zustand
- React Query
- TypeScript
- Tailwind
- SCSS Modules

## Авторизация

- JWT

## Основные возможности и страницы

- Приветственная страница с краткой информацией о сервисе
- Авторизация и регистрация
- Главная страница:
    - списки чатов по интересам
    - интересные посты из чатов
    - посты пользователей
    - рекомендации пользователей
- Поиск людей и чатов с фильтрацией по городам, возрастам и интересам
- Профиль пользователя:
    - страница с постами
    - создание поста
    - настройки
- Личные чаты
- Чаты по интересам
- Голосовые комнаты
- Песочница:
    - двухмерное воспроизведение реальных городов для прогулок и знакомств
    - генерация персонажа на основе фото пользователя (реалистичный/стилизованный) с помощью ИИ

## Примеры интересов

- программирование
- шитьё
- вязание
- баскетбол
- спорт
- языки
